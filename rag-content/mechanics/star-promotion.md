# Star Promotion System

Star promotion increases hero power through rarity upgrades.

## Rarity Tiers

| Rarity    | Stars               | Color  |
| --------- | ------------------- | ------ |
| Rare      | 1-5★                | Blue   |
| Epic      | 1-5★                | Purple |
| Legendary | 1-5★ (Silver)       | Gold   |
| Mythic    | 1-5★ (Gold/Diamond) | Red    |

## Star Progression

### Rare → Epic

- Combine Rare heroes to create Epic
- Not recommended for investment

### Epic → Legendary

- Use Epic shards/copies
- Starting point for serious heroes

### Legendary Stars (1-5 Silver)

| Stars | Shards Required | Total |
| ----- | --------------- | ----- |
| 1★    | Base            | -     |
| 2★    | 15              | 15    |
| 3★    | 20              | 35    |
| 4★    | 30              | 65    |
| 5★    | 50              | 115   |

### Awakening to Mythic

- Requires 5★ Legendary (11 total stars)
- See Awakening Guide for details

## Shard Sources

### Daily/Weekly

- Adventure Mode bosses
- Legion Boss hits
- Guild Store purchases
- Bounty rewards

### Events

- Wish for Heroes
- Special events
- Step-up rewards

### Premium

- In-app purchases
- VIP rewards

## Investment Strategy

### DO

- Focus on Legendary+ heroes
- Save universal shards for Mythic
- Prioritize S-tier heroes

### DON'T

- Invest heavily in Rare heroes
- Spread shards across too many heroes
- Use universal shards on non-Mythic heroes

## Universal Shards

**Save for Mythic heroes only!**

Universal shards can be used on any hero but are rare. Only use on:

- Natural Mythics you're actively building
- Heroes you'll awaken to Mythic

## Shard Priority

1. Main DPS carry
2. Primary tank
3. Main healer
4. Secondary DPS
5. Support heroes
