# Relic Sets Guide

Relics provide passive stats, active skills, and powerful set bonuses. Each faction has an optimal relic set.

## Relic Set Overview

| Faction    | Set Name                           | Focus                     |
| ---------- | ---------------------------------- | ------------------------- |
| **Nature** | Oath of Sacred Forest              | Survival + CC             |
| **League** | Arcane Vault                       | Control + Skill Damage    |
| **Horde**  | Dragon's Might / Crimson Slaughter | Raw Damage + Shield Break |

## Nature: Oath of Sacred Forest

### Composing Relics

| Relic                | Type       | Key Effect                                            |
| -------------------- | ---------- | ----------------------------------------------------- |
| **Undefeated Crown** | Defense #1 | "Shelter of Nature" - reduces max damage, heals team  |
| **Vineborn Bow**     | Attack #1  | "Entangle" CC - +10% damage per Nature hero (max 60%) |
| **Sacred Scroll**    | Utility    | Shield + Damage Reduction + Shield Breaker            |

### Set Bonus

- Increases duration of Entangle and Shelter effects
- Perfect synergy with Nature's sustain meta

### F2P Alternative

- Frost Diadem
- Marshal's Warhorn
- Eternal Wings

## League: Arcane Vault

### Composing Relics

| Relic                   | Type       | Key Effect                                        |
| ----------------------- | ---------- | ------------------------------------------------- |
| **Petrification Staff** | Control #1 | Petrify CC + Overall damage increase              |
| **Soul Guard Orb**      | Defense    | "Resist" - caps max damage taken, provides shield |
| **Feather of Pact**     | Buffs      | Shield value increase + Block Break               |

### Set Bonus

- +1 second to Petrify and Resist durations
- Enhances League's control-focused gameplay

### F2P Alternative

- Petrification Staff (keep this one)
- Moonstone
- Scale of Injustice

## Horde: Dragon's Might

## Acquisition

### Relic Scrolls

- **Tavern**: Dedicated relic tab
- **Guild Store**: Buy daily/weekly (mandatory!)
- **Events**: Various rewards

### Relic Shards

- **Legion Boss**: Random shards from daily hits
- **Events**: Wish for Relic, special packs

## Upgrade Strategy

### Core Principle

> **Max one relic at a time to unlock star bonuses.**

### Star Comparison

- 5★ Epic (Purple) > 1★ Legendary (Gold)
- Don't ignore Epic relics!

### Relic Hall

- Upgrade your Relic Hall building
- Grants status bonuses and power
- Unlocks additional relic slots
