# Events Guide

This guide covers major recurring events in Top Heroes and strategies for maximizing rewards.

## Weekly Event Schedule

| Day             | Event                    |
| --------------- | ------------------------ |
| Monday          | Arms Race begins         |
| Tuesday         | Guild activities         |
| Wednesday       | Legion Boss focus        |
| Thursday        | Resource gathering       |
| Friday          | PvP events               |
| Saturday-Sunday | Major events (KvK, etc.) |

_Note: Schedule varies by server and season_

## Major Events

### Kingdom War (KvK)

**Frequency**: Periodic (varies)
**Type**: Server vs Server PvP

**Preparation**:

- Max your main team before event
- Stock healing items and speedups
- Coordinate with guild

**Strategy**:

- Follow guild leadership
- Target strategic locations
- Use faction counters against enemies
- Horde excels at burst rallies
- Nature provides staying power

### Arms Race

**Frequency**: Weekly
**Type**: Point accumulation

**Strategy**:

- Save resource usage for event period
- Complete buildings during event
- Train troops during event
- Maximize point efficiency

### Guild Duel

**Frequency**: Weekly
**Type**: Guild vs Guild

**Strategy**:

- Coordinate hero bans
- Use counter-picks
- Communication is key

## Daily Events

### Legion Boss

**Frequency**: Daily
**Rewards**: Relic shards, materials

**Strategy**:

- Hit 3x daily minimum
- Use high DPS team
- Horde excels at boss damage
- Share coordinates with guild

### Bounty Hall

**Frequency**: Daily
**Rewards**: Pet eggs, materials

**Strategy**:

- Complete all 4 bounties
- Refresh for better rewards if needed
- Focus on pet eggs

## Special Events

### Pet Rush

**Rewards**: Pet Essence
**Priority**: HIGH for pet development

### Top Lord Event

**Rewards**: Premium materials
**Priority**: Participate when active

### Wish Events

**Types**: Wish for Heroes, Wish for Relics
**Strategy**: Save resources for these events

## Event Preparation Tips

### Resource Management

1. **Don't hoard permanently** - Use resources during events
2. **Plan ahead** - Know event schedule
3. **Stockpile strategically** - Save for maximum event points

### Team Preparation

1. Keep multiple faction teams ready (counter-picking)
2. Max your primary team before events
3. Have strong rally heroes for KvK

### Guild Coordination

1. Join an active guild
2. Participate in guild events
3. Follow leadership during KvK
4. Communicate hero availability
