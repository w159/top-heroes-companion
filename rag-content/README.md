# Top Heroes Game Mechanics Overview

This directory contains comprehensive documentation for all game systems in Top Heroes.

## Content Categories

### `/heroes/`

Individual markdown files for each hero (50+), including:

- Stats, skills, builds
- Team synergies and bonds
- Awakening paths
- Gear/Relic/Pet recommendations

### `/pets/`

Pet guides organized by faction:

- Eggy (League), Cactini (Nature), Flickerkit (Horde)
- Leveling strategies (focus on one pet to 120+)
- Promotion mechanics and Pet Essence

### `/relics/`

Relic sets and individual relic documentation:

- Oath of Sacred Forest (Nature)
- Arcane Vault (League)
- Dragon's Might (Horde)

### `/gear/`

Armor and gear guides:

- Knight Set (Skill Damage)
- Blood Set (HP/Tanks)
- Titan Set (Raw DPS)
- Titan Marks and Promotion

### `/events/`

Event strategies and calendars:

- Kingdom War (KvK)
- Guild Arms Race
- Legion Boss
- Seasonal events

### `/progression/`

New player and advanced guides:

- Early game priorities
- Faction selection strategies
- Castle upgrade paths
- F2P optimization

### `/mechanics/`

Core game systems:

- Faction counter system
- Awakening and Soul Stones
- Bond system (Team + Global)
- Star promotion and traits

### `/meta/`

Current meta analysis:

- Tier lists by faction
- Recommended team compositions
- PvP vs PvE builds
